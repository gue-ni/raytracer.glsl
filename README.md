# Pathtracing in a OpenGL Compute Shader

![Demo render](renders/demo.png)

## Features

-   [x] moveable camera
-   [x] depth of field
-   [x] environment sampling
-   [x] imgui integration
-   [x] triangle mesh rendering
-   [ ] object loading
-   [ ] json scene loading
-   [ ] accelaration structures 
-   [ ] physically based rendering

## Build

### Windows

```bash
mkdir build
cmake -G Ninja build/
cmake --build build/
```
